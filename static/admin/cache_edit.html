页面缓存设置
缓存开关： 相当于生成静态，可加快打开速度，但是会占用空间容量
缓存路径类型(new)： hash分布式适合无限页面模式，url方式类似静态文件路径
仅蜘蛛爬行缓存： 是否蜘蛛爬行才进行缓存
GZ压缩缓存(new)： 开启后缓存文件大小减小70%以上，但可能会加重服务器负担
页面缓存时间
首页缓存时间(index模板)： 小时，首页缓存更新时间，建议值24小时内
分类页缓存时间(list模板)： 小时，分类页缓存更新时间，建议值24小时内
内容页缓存时间(show模板)： 小时，内容页缓存更新时间，建议值48小时以上
其他页缓存时间(自定义模板)： 小时，其他页缓存更新时间，建议值48小时以上
局部缓存
tkd等缓存时间(小时)： 标题、keywords、description等缓存更新时间，0为不缓存，开启页面缓存时此选项无效
内容发布时间缓存开关： 内容页发布时间是否缓存，设置tkd缓存时才有效
外链缓存时间(小时)： 0为不缓存，如碰到开启了页面缓存刷新不变，则需要清除模板编译缓存
